# Data

Dataset preparation scripts and setup instructions.

## Scripts

| Notebook | Description |
|---|---|
| `convert_data_yaml.ipynb` | Generate `data.yaml` configuration for YOLO |
| `convert_yolo_to_coco.ipynb` | Convert YOLO annotations (`.txt`) to COCO format (`.json`) |

## Dataset: Wave-UI

**Source:** [agentsea/wave-ui](https://huggingface.co/datasets/agentsea/wave-ui)
**Annotation:** [Label Studio](https://labelstud.io/) · 16 GUI classes
**Split:** 70% train · 20% test · 10% validation

> The dataset is not included due to size. Download from Hugging Face and follow the structure in the notebooks.
