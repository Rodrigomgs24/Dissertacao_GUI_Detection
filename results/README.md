# Results

Training outputs, metrics, and comparison charts organised by architecture.

```
results/
├── yolo/              ← YOLO training outputs (auto-generated by Ultralytics)
├── faster-rcnn/       ← Faster R-CNN training curves and metrics
└── opencv/            ← OpenCV detection outputs
```

> Model weights (`.pt`, `.pth`) are excluded from git via `.gitignore`.
