# YOLO — Single-Stage Detection

YOLOv8, YOLOv10, and YOLOv11 for GUI element detection ([Ultralytics](https://docs.ultralytics.com/)).

| Notebook | Description |
|---|---|
| `train.ipynb` | Train all three YOLO variants with identical hyperparameters |
| `inference.ipynb` | Run inference on unseen test images with the best model |
